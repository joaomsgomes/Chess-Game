# Terminal Chess Game ♟️

## Overview

This project is a **simple terminal-based chess game implemented in Python**.

It was the **first programming project I ever built**, developed over a few days as an introduction to programming concepts such as game logic, control flow, and board state management.

The goal of the project was to experiment with implementing a classic board game in code and understanding how to model game rules and player interactions within a program.

While the implementation is intentionally simple, it represents an important early step in learning software development and problem solving.

---

## Features

* Chess board representation in the terminal
* Basic player move input
* Turn-based gameplay
* Piece movement validation (basic rules)
* Command-line interface

## Running the Game

Run the game directly from the terminal:

```bash id="run_chess_game"
python chess.py
```

The program will start a terminal-based chess session where players can input moves and play directly from the command line.
